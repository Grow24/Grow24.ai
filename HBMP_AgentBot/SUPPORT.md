## 🆘 Common Issues & Solutions

### Login & Authentication

#### Cannot Login / Forgot Password
1. Click "Forgot Password" on the login screen
2. Enter your registered email address
3. Check your email for password reset link
4. Follow the instructions to create a new password

#### Account Not Activated
- Check your email for the activation link
- Check spam/junk folder if not found
- Contact support if link has expired

### Chat & Conversations

#### Messages Not Sending
1. **Check Internet Connection**: Ensure you have stable internet
2. **Verify API Keys**: Go to Settings → API Keys and verify configuration
3. **Clear Cache**: Settings → Clear App Cache
4. **Restart App**: Close and reopen the application

#### AI Not Responding
- Verify your API key is valid and has credits
- Check if you've exceeded rate limits
- Try switching to a different AI model
- Check service status at your API provider's dashboard

#### Conversation History Missing
- Ensure you're logged into the correct account
- Check if conversations were archived (Settings → Archives)
- Verify sync is enabled (Settings → Sync Settings)

### File & Media

#### Cannot Upload Files
1. **Check File Size**: Maximum file size is 10MB
2. **Supported Formats**: PNG, JPG, JPEG, PDF, TXT, DOCX
3. **Check Permissions**: 
   - iOS: Settings → Privacy → Photos → Enable HBMP AgentBot
   - Android: Settings → Apps → HBMP AgentBot → Permissions
4. **Storage Space**: Ensure device has sufficient storage

#### Camera Not Working
- **Grant Camera Permission**: 
  - iOS: Settings → HBMP AgentBot → Camera → Allow
  - Android: Settings → Apps → Permissions → Camera → Allow
- Restart the app after granting permissions
- Try closing other apps using the camera

#### Photo Library Access Issues
- Enable photo library access in device settings
- For iOS: Settings → Privacy & Security → Photos → HBMP AgentBot → Select "All Photos"
- For Android: Settings → Apps → HBMP AgentBot → Permissions → Storage → Allow

### Agent Configuration

#### Agent Not Working as Expected
1. Review agent instructions and tools configuration
2. Ensure required API keys are configured
3. Check agent permissions and capabilities
4. Test with simple prompts first

#### Cannot Create New Agent
- Check if you've reached your agent limit (upgrade plan if needed)
- Verify all required fields are filled
- Ensure unique agent name

### Settings & Account

#### Cannot Update Profile
- Ensure all required fields are completed
- Check internet connection
- Try logging out and back in

#### Sync Issues Across Devices
- Enable sync in Settings → Sync Settings
- Ensure you're logged in with the same account
- Allow time for sync to complete (may take a few minutes)

#### Dark/Light Mode Not Working
- Go to Settings → Appearance
- Select desired theme
- Restart app if changes don't apply

## 🔐 Privacy & Security

### Data Privacy
- Your conversations are encrypted and stored securely
- We do not share your data with third parties
- You can export or delete your data at any time

### Delete Account
To permanently delete your account:
1. Go to Settings → Account
2. Scroll to "Delete Account"
3. Confirm deletion
4. All data will be permanently removed within 30 days

### Export Your Data
1. Go to Settings → Account → Export Data
2. Select data types to export
3. Receive download link via email
4. Download expires in 7 days

## 📱 Technical Requirements

### iOS
- **Minimum Version**: iOS 13.0 or later
- **Recommended**: iOS 15.0 or later
- **Devices**: iPhone 6s and newer, iPad Air 2 and newer

- **Recommended**: Android 11.0 or later

### Internet Connection
- Stable internet connection required
- WiFi or mobile data (4G/5G recommended)
- Minimum speed: 1 Mbps
